An open data network for creative works

Mediachain makes it simple for organizations, individuals, and developers to share and reuse information about media.

How it works

Mediachain is a protocol stack for publishing, discovering, and reusing media metadata.

Automatically resolves media to a unique identifier and translates different schemas to an interoperable format.

Efficiently explores relationships across datasets by caching information in a local graph database.

Writes all contributions to a decentralized data network powered by IPFS with blockchain guarantees.

Publish

Publish your metadata to a shared data network, where information from multiple participants is de-duplicated and made interoperable, while preserving original structure. All contributions are attributed to the publisher.

Discover

Query the network with an instance of media to retrieve relevant metadata using perceptual search, or use text search across datasets.

Collaborate

Collaborate and reuse data from multiple participants non-destructively using a git-like data structure. Search or filter statements by a contributor's cryptographic signature.

For media organizations, creators, and developers looking to share and reuse information about creative works.
