An open data network for creative works

Mediachain makes it simple for organizations, individuals, and developers to share and reuse information about media.

Architecture

Mediachain is a protocol stack for publishing, discovering, and reusing media metadata.

1

It provides metadata reconciliation to resolve media to a canonical identifier as well as translate disparate schemas to an interoperable format.

2

A local graph database serves as a write-through cache for efficient traversals of relationships and queries across datasets.

3

All contributions are persisted to a decentralized data network powered by IPFS and timestamped in the Bitcoin blockchain.

Publish

Mediachain enables publishing to a single shared data space, where information from multiple participants becomes interoperable and multiple records about the same creative work are reconciled. An identity layer preserves attribution for all contributions.

Discover

Mediachain enables querying the network with an instance of media: relevant metadata is returned using perceptual search. Additionally, efficient text queries across different datasets are supported.

Collaborate

Mediachain uses a data structure based on Git, enabling data reuse and collaboration is non-destructive and append-only. All statements are cryptographically signed by the contributor and can be searched or filtered by identity.

Contribute

Mediachain is for people looking to share and reuse information about creative works.
